# mistodon sita bot

したことを記録するmastodonのbot、[@sita@mistodon.cloud](https://mistodon.cloud/@sita)です。

heroku上にホストされています。

常時起動し通知を待ち受け、メンションがあれば処理を行って返信します。

使い方は[@sita@mistodon.cloud](https://mistodon.cloud/@sita)の固定トゥートを参照。
